项目名：遇（推箱子版）

开发者：何栋

所属小组：第8组

使用方法：
		 有编译环境的可以直接打开解决方案 遇.sln     ---路径：遇（推箱子版）\遇（推箱子版）\遇.sln 路径
		 或者打开项目 遇.vcxproj                     ---路径：遇（推箱子版）\遇（推箱子版）\遇\遇.vcxproj
		 
		 实在没有环境的就打开 遇.exe 也可以玩        ---路径：遇（推箱子版）\遇（推箱子版）\Debug\遇.exe
		  
玩法（功能介绍）：
				 每关会刷新出对应数量的宝箱、开宝箱处、石头人。
				 具体设置的生成参数：
					宝箱数：当前关卡数
					开宝箱处：与宝箱处相等
					石头人数：在原有石头人数的基础上 加上 当前关卡数*2
				 
				 每一关还会随机刷新出 “野生朋友”，玩家走近朋友距离一格以内（任何相对方位），按下f（不区分大小写）键即可与之交友，
				 之后朋友就会一直跟着玩家
				 
				 游戏的目标就是把每一关的宝箱推到开宝箱处（类似推箱子游戏），把当前关卡所有的宝箱推入指定位置即算过关
				 ，随后进入下一关
				 
				 玩家通过敲击方向键即可实现移动（注意：不是wasd哦，4399后遗症）
				 
				 玩家不能推动任何状态的朋友，同时 不能直接 推动石头人。
				 玩家可以直接推动宝箱，也可以借助宝箱 间接推动石头人
				 玩家孤身一人时只能推动一个宝箱，当身后有朋友时，玩家的力量获得加强，
				 每多一个朋友，玩家就能多推动一个单位，间接推动石头人的原理也是如此
				 
				 当玩家发现当前关卡任务过于困难时可以按下空格键，投降
				 
				 直到地图上难以再刷出单位时（也就是合理位置都被占满了），玩家通关（虽然我也不知道这样子到底能不能通关，反正我没试过）
				  
Tips：
	 “野生朋友”刷出的概率并不保持不变，随着当前拥有的朋友数的增加，刷出概率减少，所以
	 就是说：不要太贪心了捏！
	 
	 为了照顾某些非酋玩家的感受，当你需要一个朋友时，可以输入“help”（类似于作弊码），上帝会赐予你一个朋友，
	 从而一定程度上提升非酋玩家的游戏体验
	 但是上帝只会眷顾一次非酋，请不要多次使用“help”
	 
	 众所周知，玩家是无论如何都不能推动朋友的，在朋友过多的情况下，可能会出现被朋友堵角落的尴尬处境，玩家可以敲击“g”键，
	 即可实现把自己所有的朋友都聚集到自身位置的操作，优化了一定的游戏体验
	 
	 当目前关卡刷新的过于恶心时，可以敲击“r”键来重新刷新当前关卡，也是用来提升非酋的游戏体验的
